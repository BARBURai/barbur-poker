# 🎴 פוקר ברבורי תל מונד

אפליקציית ניהול קבוצת הפוקר - מעקב סטטיסטיקות, אירוחים, תובנות אישיות וגרפים.

## פיתוח מקומי

```bash
npm install
npm run dev
```

## פריסה

האפליקציה פרוסה ב-Vercel ומסתנכרנת אוטומטית עם ה-repository.

## מנוע

- React 18 + Vite
- Tailwind CSS
- Firebase Firestore (לאחסון משותף)
- Recharts (גרפים)
- Lucide React (אייקונים)
